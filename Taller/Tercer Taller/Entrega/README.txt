* El proyecto Java Maven se adjunta en formato .zip en la misma entrega; para importar:

1. Abrir Eclipse IDE for Enterprise Java and Web Developers
2. File > Import > General > Existing projects into workspace
3. Seleccionar el proyecto sepsis (previamente descomprimido)
4. Finish
5. Ajustar la versión de Java (empleado Java 11) desde:
	a. Clic derecho sobre el proyecto > Properties
	b. Java Build Path > Libraries
	c. JRE System Library > Edit 
	d. Execution environment > Seleccionar JavaSE-11 (jre)
	e. Finish > Apply and close
6. Abrir el paquete 'main'> Clase Main.java
7. Clic derecho y ejecutar como aplicación Java

* La carpeta 'Modelos' incluye los modelos en formato BPMN, PNG y SVG del proceso original,
del rediseñado para Camunda así como los formularios externos usados.
